# ROSE TaroWorks User Guide

Welcome to ROSE TaroWorks! This guide will help you use the mobile app to complete your field work.

## Understanding Your Work

### Jobs

**What is a Job?**
A job is a collection of related tasks assigned to you. Example: "Community Survey - District 1"

**Job Information:**
- **Name:** What the job is about
- **Due Date:** When it should be completed
- **Priority:** How urgent it is (High/Medium/Low)
- **Progress:** How many tasks you've completed

### Tasks

**What is a Task?**
A single unit of work within a job. Example: "Survey Household #1"

**Task Types:**
1. **View Data** - Look at existing information
2. **View Resource** - Read instructions or documents
3. **Collect Data** - Fill out a form

## Working Offline

### How Offline Works

‚úÖ **You CAN do this offline:**
- View your assigned jobs
- Read tasks and instructions
- Fill out forms
- Take photos
- Capture GPS location
- Save your work

‚ùå **You CANNOT do this offline:**
- Sync your data to the server
- Download new jobs
- View other people's data

### Offline Workflow

1. **Before Going to Field:**
   - Connect to WiFi
   - Open app
   - Sync data (green sync button)
   - Wait for "Sync Complete" message
   - Verify your jobs appear

2. **In the Field (Offline):**
   - Complete your tasks
   - Fill out forms
   - Take photos
   - All data saves to your device

3. **After Returning:**
   - Connect to WiFi
   - Open app
   - Tap sync button
   - Wait for "Sync Complete"
   - Verify "Pending Upload: 0"

## Completing Tasks

### Step-by-Step

1. **Open a Job**
   - Tap the job card
   - View list of tasks

2. **Complete Tasks in Order**
   - Start with Task 0, then 1, then 2, etc.
   - Tap a task to open it

3. **View Resource Task**
   - Read the instructions or document
   - Tap "Mark Complete" when done

4. **Collect Data Task**
   - Form will appear
   - Fill out all required fields (marked with *)
   - Tap "Submit" when complete
   - Task automatically marked complete

5. **View Data Task**
   - Review the information shown
   - Take note of important details
   - Tap "Mark Complete" when done

### Filling Out Forms

**Tips for Success:**

‚úÖ **DO:**
- Read field labels carefully
- Fill required fields (marked with *)
- Use correct format (dates, phone numbers)
- Double-check before submitting
- Save as you go (auto-saves every field)

‚ùå **DON'T:**
- Skip required fields
- Enter fake data
- Submit incomplete forms
- Rush through forms

**Field Types:**

- **Text:** Type letters and numbers
- **Number:** Type numbers only
- **Date:** Use date picker
- **Picklist:** Select from dropdown
- **GPS:** Tap "Capture Location" button
- **Photo:** Tap "Take Photo" button
- **Signature:** Sign with your finger

### Taking Photos

1. Tap **Take Photo** button
2. Allow camera permission (first time)
3. Position camera
4. Tap capture button
5. Review photo
6. Tap "Use Photo" if good, or "Retake"
7. Photo saves to form

**Photo Tips:**
- Ensure good lighting
- Hold phone steady
- Frame subject clearly
- Check photo is not blurry before accepting

### Capturing GPS

1. Tap **Capture Location** button
2. Allow location permission (first time)
3. Wait for GPS to lock (may take 10-30 seconds)
4. Location appears in form
5. If inaccurate, tap "Retry"

**GPS Tips:**
- Stand outside with clear view of sky
- Wait patiently for GPS lock
- Don't move while capturing
- If fails, try walking a few meters away

## Syncing Your Data

### When to Sync

**Always sync:**
- Before going to field (download new jobs)
- After returning from field (upload your work)
- When you have WiFi/data connection
- If instructed by your supervisor

### How to Sync

1. Connect to WiFi or mobile data
2. Open ROSE Jobs screen
3. Look for sync status at top:
   - Green WiFi icon = Online
   - Red X icon = Offline
4. Tap **Sync Now** button
5. Wait for sync to complete
6. Check results:
   - ‚úì "Sync Complete" = Success
   - ‚úó "Sync Failed" = Error (contact supervisor)

### Understanding Sync Status

**Pending Upload: 5**
- You have 5 form responses not yet uploaded
- Sync when you have connection

**Last Sync: 2 hours ago**
- Data was last synced 2 hours ago
- Sync again to get latest jobs

**Status: Offline**
- No internet connection
- You can still work, but can't sync

## Troubleshooting

### "I can't see my jobs"

**Fix:**
1. Check you're logged in correctly
2. Tap sync button
3. Wait for sync to complete
4. If still no jobs, contact supervisor

### "Form won't submit"

**Fix:**
1. Check all required fields (marked with *) are filled
2. Look for red error messages
3. Fix errors
4. Try submit again
5. If still fails, save and contact supervisor

### "Photo won't take"

**Fix:**
1. Check camera permission is allowed
2. Try closing and reopening app
3. Ensure camera lens is clean
4. Try taking photo in better light
5. If still fails, contact supervisor

### "GPS won't capture"

**Fix:**
1. Go outside with clear view of sky
2. Check location permission is allowed
3. Wait 30-60 seconds for GPS lock
4. Try walking a few meters
5. If still fails, enter approximate location and note in comments

### "Sync keeps failing"

**Fix:**
1. Check internet connection is strong
2. Try switching from WiFi to mobile data (or vice versa)
3. Close other apps
4. Restart phone
5. Try sync again
6. If still fails, contact supervisor with error message
